Directivas de Atributo

Para referirnos a las directivas de Atributos usamos [] y ()

  • [ngClass]="value1 ? 'succes' : 'error'" // Segun el valor, en este caso booleano, de la variable value1, se evalua que clase CSS aplicar
  • Ahora error es {{ error }} [ngStyle]="" // [style.background]="error? 'rgb(241, 91, 91)' : 'green'" // Permite establecer condicionales para cambiar propiedades CSS
  • (evento) = "evento()"// (click)="changeValue1()" // Evento click ejecuta la funcion changeValue1 que alterna su valor true-false

  • Notas:

    Un element HTML puede contener [ngClass] y [bgStyle] en el mismo elemento, ademas,podemos declarar ngStyle como un object y alterar mas de 1 propiedad css [ngStyle]="[object object]"

    Ejemplo: la siguiente linea de codigo altera mas de una propiedad CSS segun el valor de value1. El codigo afecta al parrafo que esta arriba // [ngStyle]=" 'border' : value1 ? '1px solid green' : '1px solid red', 'font-size' : value1? '1.5rem' : '1rem' "

Pipes

Son funciones propias de Angular, independientes de un componente y reutilizables que pueden se usada en cualquier parte por Ejemplo

Cuando utilizamos un Pipe no modificamos el valor del codigo, solo cambia la IU

Tambien podemos descargar librerias de Pipes
